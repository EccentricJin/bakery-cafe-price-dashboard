## 실행 방법

```bash
npx tsx bakery-cafe-price-dashboard.ts
```

## 포함 기능

- 10종 원자재 일별 가격 추이 (국제 선물 + 국내 도매가)
- 빵집 원가 시뮬레이션 (식빵 1개)
- 카페 원가 시뮬레이션 (아메리카노 1잔)
- 사장님을 위한 주간 인사이트
